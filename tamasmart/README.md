# Local Assets

这个目录用于放你自己准备的本地 `BIN`、固件包和备份资源。

This directory is for your own local `BIN` files, firmware packages, and backup assets.

- 仓库不会再追踪这里的镜像文件 / files here are no longer tracked by the repository
- 请自行备份、管理和命名你的资源 / manage and back up your assets yourself
- 如需公开仓库，请不要把受版权保护的资源一起提交 / do not publish copyrighted assets with the repository

你可以：

You can:

- 放入你自己的卡片 dump / place your own card dumps here
- 放入从 [Internet Archive](https://archive.org/) 等来源自行获取的资源 / store assets you obtained yourself from sources such as [Internet Archive](https://archive.org/)
- 直接在 Web UI 里上传，不一定非要放在这个目录 / upload files directly through the Web UI instead of storing them here

请自行确认资源来源、版权状态和使用合法性。

You are responsible for checking the source, copyright status, and legality of your use.
